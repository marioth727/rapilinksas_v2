# Directiva Administrativa: Footer Corporativo Rapilink

**ESTADO:** OBLIGATORIO PARA TODAS LAS VERSIONES

Para cualquier regeneración de pantalla, edición o creación de nuevas secciones en la SPA de Stitch, se DEBE mantener el siguiente esquema de Footer:

### 1. Estructura Visual
- **Tipo:** Multicolumna (5 columnas amplias).
- **Fondo Principal:** Azul Oscuro Sólido (`#1A3A5C`).
- **Barra de Cierre (Copyright):** Acción Azul (`#0072CE`) con texto blanco centrado.

### 2. Componentes por Columna
1. **Marca:** Logo Rapilink VERSION BLANCA + Slogan "Siempre Conectados" + Iconos Sociales.
2. **Sitios de Interés:** Menu de navegación (Inicio, Servicios, Planes, etc).
3. **Legal:** Cookies, Privacidad, Datos Personales, Neutralidad.
4. **Institucional:** PQRS, Trabaja con nosotros, Logo SIC.
5. **Contacto:** Teléfonos con iconos y Email corporativo.

### 3. Restricciones Técnicas
- **Logo:** Usar siempre la versión `white_logo.png` (blanco) en el footer para contraste.
- **Tipografía:** Inter (Blanco #FFFFFF).
- **Iconografía:** Línea fina monócroma.

*Nota: Cualquier edición mediante Stitch debe preservar este bloque o replicarlo fielmente si se genera una pantalla desde cero.*
